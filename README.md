# 📝 个人助手 - 多功能灵感记录工具

## 项目简介

**个人助手**是一个支持多种记录形式的灵感记录Web应用。灵感来得很突然，可能是一个画面、一段旋律、一个想法，如果不及时记录就会忘记。现有备忘录工具大多只支持文字，无法完整记录复杂灵感。本工具支持**音频、视频、图片、文字、链接、涂鸦**等六种记录形式，让每个灵感都能被妥善保存。

## 目标用户

- 记性不好容易忘事的人
- 创意工作者（设计师、音乐人、作家等）
- 学生和职场人士
- 喜欢记录生活感受的用户

## 解决痛点

- ❌ **灵感易流失**：传统备忘录打开慢、操作繁琐
- ❌ **记录形式单一**：无法记录声音、画面等复杂灵感
- ❌ **分类混乱**：不同类型灵感分散在不同工具中
- ❌ **检索困难**：缺少标签分类和搜索功能

## 核心功能

### 📝 文字记录
- 支持标题、内容、标签记录
- 快速保存文字灵感
- 自动记录保存时间

### 🎵 音频录制
- 浏览器实时录音功能
- 支持麦克风音频采集
- 录音预览和回放

### 📷 图片捕捉
- 上传图片记录画面灵感
- 支持图片预览
- 本地存储图片数据

### 🔗 链接收藏
- 保存网页链接
- 一键访问收藏链接
- 支持链接标题备注

### 🎨 随手涂鸦
- 画布绘图工具
- 支持颜色选择
- 支持笔刷大小调节
- 清空画布功能

### 🔍 搜索筛选
- 关键词搜索功能
- 分类筛选（文字、音频、图片、链接、涂鸦）
- 标签管理
- 时间排序显示

### 💾 数据管理
- 本地localStorage存储
- 数据持久化，刷新不丢失
- 删除灵感功能
- 实时统计已保存数量

## 技术特点

- ✅ **纯前端实现**：无需后端服务器
- ✅ **响应式设计**：适配手机和电脑
- ✅ **深色主题**：现代化UI设计
- ✅ **数据持久化**：localStorage本地存储
- ✅ **浏览器API**：录音、文件上传、Canvas绑定

- 📸 **留住生活感受**：记录美好瞬间，形成成长档案
- 💪 **提升效率**：作为多功能备忘录，辅助学习和工作

## 使用方法

1. 打开网站 `personal-assistant-app.html`
2. 点击左侧对应的记录按钮（文字、音频、图片等）
3. 在右侧面板输入/上传内容
4. 添加标签（可选）
5. 点击"保存"按钮
6. 所有灵感会显示在"我的灵感库"区域
7. 可使用搜索框查找灵感
8. 可按类型筛选查看
